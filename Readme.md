# 🎨 Portfolio Personnel

Mon site portfolio personnel codé from scratch en HTML/CSS/JavaScript vanilla.

## 🌐 Voir le site

🔗 **[https://martinyluca-eng.github.io/Portfolio/](https://martinyluca-eng.github.io/Portfolio/)**

## ✨ Fonctionnalités

- Design responsive (desktop & mobile)
- Animations au scroll
- Effets interactifs au survol
- Dégradés colorés et polices personnalisées

## 🛠️ Technologies

- **HTML5** — structure
- **CSS3** — design, animations, responsive
- **JavaScript vanilla** — animations au scroll (IntersectionObserver)
- **Google Fonts** — typographies Syne & DM Mono
